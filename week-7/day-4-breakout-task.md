#### Goal

Style the site and improve the layout.

#### Steps

1. Link your CSS file on every page.
2. Use classes to keep styles consistent.
3. Use flexbox at least once for a real layout.
4. Test hover styles, spacing, and readability as you go.
5. If you finish early, improve one part of the site that still feels plain.

#### Extra practice if you want

- Add clearer hover styles to links or buttons.
- Clean up repeated styles by turning them into classes.
