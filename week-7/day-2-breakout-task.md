#### Goal

Add content and structure to every page in your Web Lab project.

#### Steps

1. Make sure each page has a clear heading and main content.
2. Add images with alt text where they help the site.
3. Keep page names, text, and sections organized.
4. Test your navigation on every page.
5. If you finish early, fix broken links or clean up spacing.

#### Extra practice if you want

- Add one more section to the home page.
- Improve one image caption or alt text.
