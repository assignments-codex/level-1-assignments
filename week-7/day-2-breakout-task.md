#### Goal

Add real content and connect pages with navigation.

#### Steps

1. Add the same navigation on every page.
2. Test that all links work between pages.
3. Add real headings and paragraph content to each page.
4. Add at least one image to each page with meaningful alt text.
5. Improve your wireframe based on the work you have done so far (update layout, sections, or navigation if needed).
6. Make sure header, main, and footer are present and used correctly on every page.
