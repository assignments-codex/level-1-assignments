#### Goal

Build the core logic for your App Lab project.

#### Steps

1. Add events for the main buttons or inputs.
2. Use at least one variable to track state.
3. Start building the decision logic for the app.
4. Test the app often so you catch bugs early.
5. If you finish early, add clearer text or visual feedback.

#### Extra practice if you want

- Add a reset button.
- Move repeated code into a named function.
