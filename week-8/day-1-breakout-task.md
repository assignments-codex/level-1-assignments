#### Goal

Plan a fresh App Lab project inspired by Lesson 11 or a new idea.

#### Steps

1. Pick a project idea. You may use Lesson 11 for inspiration, but do not clone a lesson project.
2. Decide what the app does, what screens you need, and what the main user actions will be.
3. Start a fresh App Lab project and build the screen layout.
4. If you finish early, wire one simple interaction.

#### Extra practice if you want

- Sketch the screens before you build them.
- Write down the main events your app will need.
