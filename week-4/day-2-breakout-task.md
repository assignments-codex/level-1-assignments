#### Goal

Finish Lesson 3 Multi-screen Apps.

#### Steps

1. Open Lesson 3 in App Lab.
2. Build the screens from class and make sure navigation works in both directions.
3. Test every button so you can move through the app without getting stuck.
4. Keep ids readable so you know which screen or button you are using.
5. If you finish early, add one extra screen or one extra button to practice navigation.

#### Extra practice if you want

- Rename one unclear id so it is easier to understand.
- Add one more path through the app and test it.
