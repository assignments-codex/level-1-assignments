#### Goal

Use App Lab to practice variables, functions, and passing values.

#### Steps

1. Open your Color Sleuth project or start a fresh App Lab project.
2. Create one named function that updates something on the screen.
3. Call that function from an event handler.
4. Use at least one variable to track state such as score, round, or turn.
5. If you finish early, rename ids and variables so they are easier to read.

#### Extra practice if you want

- Add a reset button that calls one of your functions.
- Move repeated code into a named function.
