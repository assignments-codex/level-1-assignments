#### Goal

Finish your App Lab skills checkpoint and clean up your code.

#### Steps

1. Test every screen, button, and result in your app.
2. Make sure text, buttons, and outputs are easy to understand.
3. Move repeated code into a named function if it helps.
4. Fix small bugs before you add anything new.
5. If you finish early, add one small improvement such as clearer feedback or a reset button.

#### Extra practice if you want

- Rename one unclear variable or id.
- Add one more test case and make sure the app still works.
