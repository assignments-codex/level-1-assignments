#### Goal

Use App Lab to practice user input, variables, and simple logic.

#### Steps

1. Start from a fresh App Lab project or continue one from yesterday.
2. Read input from the user and show a result on the screen.
3. Use a variable to store something that changes.
4. Add simple logic so the result changes based on what the user does.
5. If you finish early, try storing a few values in an array and show one on button click.

#### Extra practice if you want

- Add a second input or a second button.
- Improve the text so the user always knows what to do next.
